# Computer-Accessories-website
I have built an interactive inventory management system to track and analyze computer accessories performance across different categories, pricing tiers, and stock availability statuses.
✨ Key highlights :-

Tracked key inventory metrics such as :-
 * Total products
 * Number of available items
 * Out-of-stock products
 * Individual item pricing

Analyzed inventory performance based on :-
 * Product type (Category-wise analysis including headphones, speakers, keyboards, etc.)
 * Availability status
 * Price distribution
Discovered top-performing product categories contributing to the overall stock layout.
Observed highest-performing item entries across different statuses.
Evaluates product types to determine which contributes the most to total value.

This project helped me understand how clean UI data structures are used for real inventory decision-making.
🛠️ Tools used :- HTML5, CSS3, and JavaScript to construct a clean, responsive front-end dashboard with native dynamic filtering and table management features.
